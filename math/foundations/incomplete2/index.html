ことが大切です。今回は再帰的関数について、基本的な定理を証明します。 突然不完全性定理とは関係がなくなったかのように思われますが、 実はこれがかなり重要になります。 我々は論理式を自然数にコーディングしたいわけですが、そのためには 様々な複雑な関数が必要になります。しかしペアノ算術の体系には 足し算と掛け算くらいしかありません。 論理式、証明、といった概念を算術化するために必要な関数は どうするのでしょうか。 実は、再帰的な関数ならペアノの言葉で"表現"できることを示します。

定義 (原子再帰的関数)

原子再帰的関数は次のように定義される。
1. 次に示す初期関数と呼ばれるは原子再帰的である。

$S(x)= x+1$
$E_i^n(x_1,...,x_n) = x_i$
$C_q^n(x_1,...,x_n) = q$

2. 関数$h(x_1,...,x_m)$と$g_1(x_1,...,x_n),...,g_m(x_1,...,x_n)$が原子再帰的であるとき

次のように得られる$f(x_1,..,x_n)$も原子再帰的
$f(x_1,..,x_n) = h(g_1(x_1,...,x_n),...,g_m(x_1,...,x_n))$

3. 関数$h(x_1,...,x_n,x_{n+1},x_{n+2})$と$g(x_1,...,x_n)$が原子再帰なら

次のような関数$f(x_1,...,x_n,x_{n+1})$も原子再帰的
$f(x_1,...,x_n,0)=g(x_1,...,x_n)$
$f(x_1,...,x_n,k+1) = h(x_1,...,x_n,k,f(x_1,...,x_n,k))$　　　$(k=0,1,2,...)$

4. 3.の特殊な場合で、$n=0$の場合も許す。この時は$g$の部分は定数とする。

$f(0) = q$
$f(k+1) = h(k,f(k))$

さらに、1.～4.に加えて、次の5.も用いて得られるような関数のことは、再帰的であるという。

定義 (再帰的関数)

再帰的関数は次のように定義される。
1. 次に示す初期関数と呼ばれるは再帰的である。

$S(x)= x+1$
$E_i^n(x_1,...,x_n) = x_i$
$C_q^n(x_1,...,x_n) = q$

2. 関数$h(x_1,...,x_m)$と$g_1(x_1,...,x_n),...,g_m(x_1,...,x_n)$が再帰的であるとき

次のように得られる$f(x_1,..,x_n)$も再帰的
$f(x_1,..,x_n) = h(g_1(x_1,...,x_n),...,g_m(x_1,...,x_n))$

3. 関数$h(x_1,...,x_n,x_{n+1},x_{n+2})$と$g(x_1,...,x_n)$が再帰的なら

次のような関数$f(x_1,...,x_n,x_{n+1})$も再帰的
$f(x_1,...,x_n,0)=g(x_1,...,x_n)$
$f(x_1,...,x_n,k+1) = h(x_1,...,x_n,k,f(x_1,...,x_n,k))$　　　$(k=0,1,2,...)$

4. 3.の特殊な場合で、$n=0$の場合も許す。この時は$g$の部分は定数とする。

$f(0) = q$
$f(k+1) = h(k,f(k))$

5. 自然数の関係$R(x_1,...,x_n,y)$があるとき、$R(x_1,...,x_n,y)$を満たすような$y$のうち最小 ものを$\mu y R(x_1,...,x_n,y)$と書くことにする。$g(x_1,...,x_n,y)$が原子再帰的の時、 次で定義される$f(x_1,...,x_n)$も原子再帰的。

$f(x_1,...,x_n) = \mu y (g(x_1,...,x_n,y)=0)$

通常の足し算$f(x,y)=x+y$や$f(x,y)=x \cdot y$などは原子再帰的であることが分かります。
定義 (再帰的関係、原子再帰的関係)

自然数の関係$R(x_1,...,x_n)$について、$R(x_1,...,x_n) \leftrightarrow \phi(x_1,...,x_n)=0$ となる関数$\phi$が再帰的であるとき、$R$は再帰的であるという。 $\phi$が原子再帰的なら$R$も原子再帰的であるという。

具体例としては、$R(p) \equiv$ $p$は素数である、は原子再帰的です。
定理

$R(x_1,...,x_n)$と$Q(x_1,...,x_n)$が再帰的(原子再帰的)の時、次の関係も再帰的(原子再帰的)。

$\lnot R$
$R \land Q$
$R \lor Q$
$R \implies Q$
$\exists x_1 < y R(x_1,...,x_n)$
$\forall x_1 < y R(x_1,,..,x_n)$

さらに、$R$が原子再帰的なら、次の関数$f$も原子再帰的。

$f(x_1,...,x_n) = \mu x_1 < y R(y,...,x_n)$

証明は難しくありません。さて、これらの性質と、足し算掛け算が原子再帰的であることを 使って、コーディングに必要な関数が原子再帰的であることを示していくのです。
